Gaussian Process¶

El proceso Gaussiano se trata de un modelo de regresin no paramtrica cuyo objetivo es encontrar una relacin sencilla y til entre X e Y para poder explicar y predecir el valor de la Y a partir de X. El modelo de regresin no pramtrica plantea una estimacin de tipo local, que resulta ms conveniente en algunas configuraciones de datos.

Construimos el modelo:¶

In [9]:

def mean_absolute_percentage_error(y_train, y_pred_validation): 
    return np.mean(np.abs((y_train - y_pred_validation) / y_train)) * 100
In [15]:
def evaluacion_prediccion(consumo_energia, y_pred_validation):
    results = pd.DataFrame({'r2_score':r2_score(consumo_energia, y_pred_validation),
                           }, index=[0])
    results['mean_absolute_error'] = mean_absolute_error(consumo_energia, y_pred_validation)
    results['median_absolute_error'] = median_absolute_error(consumo_energia, y_pred_validation)
    results['mse'] = mean_squared_error(consumo_energia, y_pred_validation)
    results['msle'] = mean_squared_log_error(consumo_energia, y_pred_validation)
    results['mape'] = mean_absolute_percentage_error(consumo_energia, y_pred_validation)
    results['rmse'] = np.sqrt(results['mse'])
    return results

En este modelo podemos ver que el coeficiente de determinacin R cuadrado, que explica la cantidad de varianza explicada de los datos, es sustancialmente bajo.

El mean_absolute_error mide la magnitud media de los errores en un conjunto de pronsticos, sin tener en cuenta su direccin. Mide la precisin de las variables continuas y el rmse, que es el error cuadrtico medio que es la diferencia entre la previsin y los valores observados correspondientes al cuadrado.

Ambos se pueden utilizar juntos para diagnosticar la variacin en los errores en un conjunto de previsiones. El cuadrtico siempre ser mayor o igual al MAE; cuanto mayor sea la diferencia entre ellos, mayor ser la varianza en los errores individuales de la muestra.

La diferencia media entre la prediccin del consumo de energa y la observada fue de 0.07

El coeficiente R cuadrado quizas es ms adecuado para indicar la varianza explicada en modelos lineales o el proceso gausiano quizas no se adecue a explicar este tipo de datos, por lo que podria funcionar de manera ms precisa en otros mbitos
