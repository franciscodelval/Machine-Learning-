Modelo Arima¶

Modelo estadstico que utiliza variaciones y regresiones de datos estadsticos con el fin de encontrar patrones para una prediccin hacia el futuro. Se trata de un modelo dinmico de series temporales, es decir, las estimaciones futuras vienen explicadas por los datos del pasado y no por variables independientes.

    if results.aic < best_aic:
        best_aic = results.aic
        best_pdq = param

El error cuadrtico medio (RMSE) es la desviacin estndar de los residuos ( errores de prediccin ). Los residuos son una medida de qu tan lejos estn los puntos de datos de la lnea de regresin; RMSE es una medida de la dispersin de estos residuos. En otras palabras, le dice qu tan concentrados estn los datos alrededor de la lnea de mejor ajuste.

El error de porcentaje absoluto medio (MAPE), tambin conocido como desviacin de porcentaje absoluto medio (MAPD), es una medida de precisin de prediccin de un mtodo de pronstico en estadstica, por ejemplo, en estimacin de tendencias, tambin utilizado como funcin de prdida para problemas de regresin en Machine Learning.
